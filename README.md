# Furniture Image Library

室內設計參考圖庫，供 AI 圖像生成工具（Gemini/Nano Banana）使用。

## 使用方式

圖片 URL 格式：
```
https://raw.githubusercontent.com/ZhiQiYang/furniture-image-library/main/[category]/[item].jpg
```

## 資料夾結構

```
furniture-image-library/
├── sofa/              # 沙發
├── chair/             # 椅子
├── table/             # 桌子
├── bed/               # 床
├── storage/           # 收納
├── lighting/          # 燈具
└── decor/             # 裝飾品
```

## 圖片規格

- 格式：JPG 或 PNG
- 建議尺寸：800x800 ~ 1200x1200
- 背景：白色或透明（去背更佳）
- 命名：`品牌_產品名.jpg`（如 `ikea_kivik.jpg`）

僅供個人學習與測試使用。商業使用請確認原始圖片授權。
